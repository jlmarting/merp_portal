# Tablas de Críticos MERP en Formato JSON

### Tablas de Críticos MERP en Formato JSON

Aquí tienes la estructura **JSON** completa que integra las tablas de críticos **CT-1 a CT-10**, extrayendo íntegramente las descripciones y rangos de las fuentes del manual de MERP.

Esta estructura está optimizada para ser consultada por aplicaciones o herramientas de juego, incluyendo los modificadores de gravedad estándar y los específicos para armas contra grandes criaturas [1-6].
