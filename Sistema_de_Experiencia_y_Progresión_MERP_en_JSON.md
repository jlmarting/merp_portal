# Sistema de Experiencia y Progresión MERP en JSON

### Sistema de Experiencia y Progresión MERP en JSON

Aquí tienes la estructura **JSON** completa que integra las tablas de experiencia y progresión de niveles **ET-1 a ET-5**, extraídas de las páginas 22 a 24 del manual y tus notas técnicas.

Esta estructura incluye no solo los valores numéricos, sino también las reglas de cálculo para niveles superiores a 10 y los multiplicadores situacionales.
