# Tablas de Pifias y Fallos de MERP JSON

### Tablas de Pifias y Fallos de MERP JSON

Aquí tienes la estructura **JSON** completa que integra las tablas de fallos y pifias **FT-1 a FT-4**, extraídas íntegramente de las páginas 166 y 167 del manual [1].

Esta estructura incluye todas las descripciones de los efectos, los rangos de tirada y los modificadores específicos para cada tipo de acción [2-6].
