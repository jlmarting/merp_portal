# Tablas JSON de Ataque para Criaturas MERP

### Tablas JSON de Ataque para Criaturas MERP

Aquí tienes la estructura JSON para las tablas de ataque de criaturas (**AT-5 y AT-6**), basadas íntegramente en los valores de las páginas 164 y 165 del manual [1-4].

Estas tablas son particulares porque sus resultados máximos de daño y gravedad de críticos están limitados por el tamaño del animal (Pequeño, Mediano, Grande o Enorme) [5, 6].

### Notas sobre los datos:

**SM***: Indica que si la tirada de dados (antes de sumar la Bonificación Ofensiva) es 01 o 02, el ataque falla automáticamente [1, 2].

**Escalado por Tamaño**: A partir de una tirada total de 105, los resultados están limitados. Por ejemplo, un animal pequeño nunca podrá infligir más daño que el indicado en su fila de "Max. resultados" aunque la tirada sea de 150 [5].

**Tipos de Crítico**: Para **AT-5**, el tipo de daño (Tajo, Perforación, etc.) se define en la tabla **CST-2** según el animal. Para **AT-6**, suelen ser críticos de **Desequilibrio (CT-4)** o **Presa (CT-5)** [10].
